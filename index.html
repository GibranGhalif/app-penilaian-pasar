SPI 300 - Penilaian Real Properti

Selamat Datang, Admin!

Ini adalah dashboard Sistem Penilaian Aset berdasarkan Standar Penilaian Indonesia (SPI 300). Gunakan alat bantu ini untuk melakukan estimasi nilai pasar properti dengan cepat dan terstruktur.

Pendekatan Data Pasar

Metode perbandingan data pasar yang membandingkan aset dengan properti sejenis.

Pendekatan Biaya

Kalkulasi nilai tanah ditambah biaya penggantian baru (RCN) dikurangi penyusutan.

Pendekatan Pendapatan

Metode kapitalisasi pendapatan untuk properti yang menghasilkan income (sewa/bisnis).

Informasi SPI 300

Penilaian Real Properti bertujuan untuk memberikan opini nilai ekonomi atas hak kepemilikan properti.

Nilai Pasar didasarkan pada prinsip Highest and Best Use (HBU).

Inspeksi lapangan wajib dilakukan untuk memverifikasi fisik dan kondisi aset.

Tanggal penilaian (Valuation Date) sangat krusial dalam menentukan asumsi pasar.

Cara Menggunakan Firebase (Gratis)

  1. Buka Firebase Console
  2. Buat project baru atau pilih project yang ada
  3. Klik "Realtime Database" → "Create Database"
  4. Pilih "Start in test mode" untuk pengembangan
  5. Klik gear icon → "Project settings" untuk mendapatkan config
  6. Copy API Key, Auth Domain, Database URL, Project ID, Storage Bucket
  7. Paste ke form konfigurasi di atas

F. CATATAN DAN ASUMSI

1. Penilaian ini dilakukan berdasarkan Standar Penilaian Indonesia (SPI) khususnya SPI 300 tentang Penilaian Real Properti.

2. Nilai pasar adalah estimasi nilai pada tanggal penilaian dalam transaksi yang wajar antara penjual dan pembeli yang berminat.

3. Penyesuaian dilakukan berdasarkan perbedaan karakteristik fisik, lokasi, legalitas, dan waktu transaksi.

4. Hasil penilaian ini hanya berlaku untuk tujuan sebagaimana disebutkan dan tidak dapat digunakan untuk tujuan lain.

5. Nilai yang tercantum merupakan nilai estimasi dan dapat berbeda dengan nilai transaksi aktual.
